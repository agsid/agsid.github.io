Allied Work

Allied Work allows user to see where their robots will go on the game mat with the built in visualizer. This is very helpful because many times teams are using trial and error to reach where they want to on the game mat. Not only can teams do this they can also code with ease. The Allied Workspace allows users to code in its native language called Allied Work. Allied Work is made to be simple so everybody can code their robot. When the you are done coding you path you can press the run button to see the path of the robot then copy the python code from the built in python generator.

Allied AutoCorrection

Allied AutoCorrection is a beautiful and helpful function for all FLL teams. The function provides precision in all aspects. Imagine you are walking straight and you trip. Now you will get back in the same Line you were walking in, and walk until you reach your final destination. Allied AutoCorrection is here to help the robot to do the same. One of the sensors used is the gyro sensor. A gyro sensor helps track the robots position in a graph. The gyro sensor helps with the x, y, and z plane. This helps us to see if the robot has encountered an error.

How are they related

You may be asking how a autocorrection software and an auto correction function related. Well in the Allied Work editor whenever you code a command for movement such as drive it takes the Allie auto correction function and uses it to give accurate movements
