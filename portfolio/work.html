AlliedWork – Advanced LEGO SPIKE Editor

The futuristic editor designed to code LEGO SPIKE Hub robots. Write, test, and visualize your programs with precision and speed.

Key Features

Real-Time Visualization

Watch your LEGO SPIKE Hub robot’s movements live on a virtual mat as you code. Instantly see how your commands affect the robot's path.

Multi-File Projects

Create, organize, and manage multiple files within the editor. Split complex programs into modules for better readability.

Robot Control Commands

Access advanced SPIKE Hub commands like drive, turn, arm control, wait, and sensors integration. Build precise robot behaviors easily.

Robot Commands

drive

Moves the robot forward or backward by a set distance (in inches).

drive(24)

turn

Rotates the robot by a given angle in degrees.

turn(90)

speed

Sets the robot’s movement speed (0–100).

speed(80)

wait

Pauses movement for a specified number of seconds.

wait(2)

arm_speed

Controls the speed of the robot’s arm motor.

arm_speed(60)

arm

Moves the arm up or down by a specific degree value.

arm(-45)
