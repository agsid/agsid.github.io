Commands

Allied Work Docs commands you can use to control your robot.

Drive

The drive command moves the robot forward or backward by a specified distance. 

drive forward 10
drive backward 5

Turn

The turn command rotates the robot in place by the given degrees. 

turn right 90
turn left 45

Arm

The arm commands control the robotic arm. 

arm lift 3
arm lower 2
arm grab
arm release

Extra Commands

Additional utility commands to enhance robot interaction.

  • wait <seconds> — pauses execution
  • beep — activates sound buzzer
  • lights on/off — toggle robot LEDs
