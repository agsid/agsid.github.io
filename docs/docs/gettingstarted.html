Getting Started

Welcome to the Allied Work Docs — learn how to set up, write, and run your first program with ease.

Introduction

The Allied Work Docs system provides an intuitive way to create and automate your robot tasks with readable, efficient code. This guide will walk you through the basics to get you started.

Why?

Traditional robot programming often involves unnecessary complexity. Our syntax was designed to simplify that — making code closer to how you think. Whether you’re new or experienced, you’ll find clarity and control.

What is it?

Allied Work Docs is a modular command framework for your robotics projects. It lets you chain logical and physical actions such as movement, arm control, and environment response using readable, structured commands.

  • Works on any Allied-compatible board
  • Designed for automation, robotics, and educational control systems
  • Includes built-in autocorrection and syntax validation

How to Get Started

  1. Download the Allied Work Docs environment.
  2. Connect your Allied-compatible robot controller.
  3. Open the editor and create your first .awd file.
  4. Write your commands using the guide below.
  5. Click Run to execute directly on your device.

Syntax and Structure

The Allied syntax follows a natural-language style, where each command is readable and minimal. 


drive forward 10
turn right 90
arm lift 3
wait 2
drive backward 5

Each instruction is executed sequentially. You can also chain conditions, loops, and advanced motion patterns for dynamic control.
